The current PC value plus three is pushed onto the stack. The MSB is loaded with $00 and the LSB is loaded with imm8.

Syntax

RST imm8

Allowed Instructions

RST $00
RST $08 ;rOP1ToOP2
RST $10 ;rFindSym
RST $18 ;rPushRealO1
RST $20 ;rMove9ToOP1
RST $28 ;rBCALL
RST $30 ;rFPAdd
RST $38

Effects

All flags unaffected.

Uses

T-States

11 t-states
