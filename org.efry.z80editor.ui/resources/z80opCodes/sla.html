Shift left arithmetic. m=m*2

Syntax

sla op8

Allowed Instructions

SLA A
SLA B
SLA C
SLA D
SLA E
SLA H
SLA L
SLA (HL)
SLA (IX+n)
SLA (IY+n)

Effects

S and Z by definition, H and N reset, C from bit 7, P/V set if result is even.

Uses

T-States

r denotes 8-bit register.

r 8
(hl) 15
(ix+X) 23
(iy+X) 23
